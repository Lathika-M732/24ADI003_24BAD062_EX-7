SCENARIO 1 – CLUSTERING USING K-MEANS
DATASET LINK:https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python
     This project focuses on reducing the complexity of high-dimensional data using Principal Component Analysis (PCA). The objective is to transform the original set of features into a smaller number of uncorrelated variables, known as principal components, while preserving most of the important information (variance) in the dataset.By applying PCA, the data becomes easier to visualize, computationally efficient, and less prone to overfitting. This technique helps uncover the most significant patterns in the data and is commonly used as a preprocessing step in various machine learning models.

SCENARIO 2 – CLUSTERING USING GMM
DATASET LINK:https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python
      This project focuses on performing customer segmentation using the Gaussian Mixture Model (GMM) clustering technique. Unlike K-Means, GMM is a probabilistic model that assumes data is generated from a mixture of multiple Gaussian distributions. Each data point is assigned a probability of belonging to each cluster rather than being strictly assigned to just one.The objective is to capture complex and overlapping patterns in the data, providing more flexible and realistic clustering. This helps in better understanding customer behavior and enables businesses to design personalized marketing strategies and improve decision-making.
